# englishApp
Web app based on PHP &amp; AngularJS, making in free time for hobby and to improve skills.

Assumptions of app:
- app created to help learning foreign language (eng)
- simple, user-friendly and responsive design
- logged user can add words with meaning in native language,
- can create and add tags (categories)
- can see all list of words, search particular word, filter by tag
- can generate test - to check skills
- tests can be created with defined options like tags, all words / this words, which user doesn't "pass", how many words will be in one test* 
- after filling test, correct answers (words) will be checked as learned
- user can generate table to print (for easily learning)*
